= Replace a NIC in the SGF6112 appliance
:icons: font
:imagesdir: ../media/

[.lead]
The SGF6112 has three 2-port NICs. If one of them fails, use these procedures to replace it.

.What you'll need

* You have xref:locating-sgf6112-in-data-center.adoc[physically located the appliance] with the NIC to be replaced.

* You have unpacked the replacement NIC and ensured that it is the same model as the one you are replacing. 

.About this task

The figure shows the locations of the NICs for the SGF6112. The NICS are accesible after the appliance cover is removed.

image::../media/sgf6112_power_sxxxxxxxupplies.png[SGF6112 with cover removed]
